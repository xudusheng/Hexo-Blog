# hexo-theme-stellar

Elegant and powerful theme for Hexo.

## Getting Started

Check your environment:

```yaml
Hexo: 5.3.0
hexo-cli: 4.2.0
node.js: 14.15.4 LTS
npm: 6.14.10 LTS
```

Edit your `_config.yml`:

```yaml
theme: stellar
```

Install Stellar in terminal:

```bash
npm i hexo-theme-stellar
```
